Native HTML5 Attributes

You'll want to be familiar with some of the newer attributes as well and ensure that they are cross browser compliant if you use them in your production code

Placeholder

 <input type="text" placeholder="..">

Maxlength

 <input type="text" maxlength="100">

Minlength

 <input type="text" minlength="10">

Pattern

 <input type="text" pattern="/$2/*`">
